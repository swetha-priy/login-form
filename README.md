# 🚀 Registration Form

A modern, responsive, and interactive **User Registration Form** built using **HTML, CSS, and JavaScript**. This project demonstrates form validation, password strength checking, password visibility toggling, and a clean user-friendly interface without using any external libraries or frameworks.

---

## 📌 Features

✅ Responsive design for desktop and mobile devices

✅ First Name & Last Name inputs

✅ Username validation

✅ Email validation using regular expressions

✅ Password and Confirm Password validation

✅ Password visibility toggle

✅ Real-time password strength indicator

✅ Terms & Conditions checkbox validation

✅ Forgot Password functionality

✅ Success message banner after account creation

✅ Clean and modern UI with SVG icons

---

## 🎯 Validation Rules

| Field | Validation |
|---------|------------|
| First Name | Required |
| Last Name | Required |
| Username | Minimum 3 characters |
| Email | Must be a valid email format |
| Password | Minimum 8 characters |
| Confirm Password | Must match password |
| Terms & Conditions | Must be checked |

---

## 🔒 Password Strength Meter

The password strength indicator evaluates:

- Minimum length of 8 characters
- Bonus for 12+ characters
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

Strength levels:

- Weak
- Fair
- Good
- Strong
- Very Strong

## ⚙️ Functionalities

### Password Visibility Toggle

Users can show or hide their password using the eye icon.

### Real-Time Validation

Input fields are validated instantly and display error messages when invalid.

### Forgot Password

Clicking the **Forgot Password?** link displays a password reset notification.

### Success Notification

When all validations pass and Terms & Conditions are accepted:

- Form fields are cleared
- Success banner is displayed
- Banner automatically disappears after 4 seconds
